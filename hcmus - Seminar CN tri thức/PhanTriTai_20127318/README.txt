1) Mở Visual Studio Code để chạy code

2) Tải IntelliJ IDEA (https://www.jetbrains.com/idea/download/?section=windows)

Trong menu chính, chọn File → Open và chọn tệp RabbitMQ Client pom.xml. Bấm vào nút Open.

Chạy ứng dụng bằng cách nhấp chuột phải vào lớp chính RabbitMQClient và chọn Chạy 'RabbitMQClient'.

3) Điều hướng đến RabbitMQ Client → http://localhost:8080 trong trình duyệt của bạn.
